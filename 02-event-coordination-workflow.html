📅 Event Coordination Workflow

Document Type: Operations Workflow
Category: Event Coordination
Version: 1.0
Last Updated: February 2025
Author: Margaret Muriuki, Client Services Coordinator

📌 Overview

This document provides a structured workflow for planning and executing company events, including client meetings, webinars, and internal engagements.

Applicable Events:

  • Client onboarding sessions
  • Quarterly business reviews (QBRs)
  • Internal team meetings and trainings
  • Webinars and virtual workshops

🗓️ Phase 1: Event Planning (4–6 Weeks Before)

Define the Event

  • Confirm purpose, audience, and outcomes
  • Determine format: in-person / virtual / hybrid
  • Set date and time
  • Create SharePoint event folder

Planning Tracker

Item Owner Deadline Status
Venue/Platform booking Coordinator Week 1 Done
Agenda draft Manager Week 2 In Progress
Invitations sent Coordinator Week 2 Pending

📧 Phase 2: Communication & Invitations

Send Invitations

  • Draft invitation email
  • Send via Outlook with calendar invite
  • Log invitees in HubSpot

Track RSVPs

  • Update RSVP tracker
  • Send reminders
  • Confirm final attendees

🔧 Phase 3: Pre-Event Preparation

Logistics

  • Confirm venue/platform
  • Ensure materials are ready
  • Distribute agenda

Technical Setup

  • Submit ServiceNow request
  • Test Teams features
  • Confirm presenters

🎯 Phase 4: Event Day Execution

Before the Session

  • Log in 30 minutes early
  • Test audio/video
  • Prepare attendance sheet

During the Session

  • Welcome attendees
  • Monitor chat
  • Take minutes

Technical Issues

Issue Action
Presenter cannot connect Use backup presenter
Audio failure Switch to phone audio
Recording stops Restart recording

📝 Phase 5: Post-Event Follow-Up

  • Send follow-up email
  • Share minutes and slides
  • Log action items
  • Upload recording

🛠️ Tools Used

  • Microsoft Outlook
  • Microsoft Teams
  • SharePoint
  • HubSpot
  • ServiceNow
  • Excel
